# Expo Go Setup

## Installation Process

1. Opened the **Google Play Store** on my Android device.
2. Searched for **Expo Go**.
3. Downloaded and installed the Expo Go application successfully.

## Account Setup

1. Launched the Expo Go app.
2. Navigated to the **left-hand side menu** and clicked on **Sign Up**.
3. Entered my details (name, email, password) to create a new account.
4. Logged in successfully to Expo Go.
